What are HTML, CSS, and JavaScript




HTML, CSS, JAVA : They are programming language to communicate with software.
They are used to create website for better user experience.




HTML


HTML stands for Hyper Text Markup Language which is the core of a web site.
It works by using tags to identify a variety of contents and it also serve purposes to the webpage as well.
Therefore, HTML provides a page structure which can be adjusted and magnified by using CSS and JavaScript.




CSS


CSS stands for Cascading Style Sheets.
It is style information functioning for presentation, formatting, and layout.
In other words, CSS controls how the website will look like.




JavaScript


A scripting language to create a more dynamic page.
Nowadays, all modern webpage support JavaScript.

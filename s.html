What are HTML, CSS, and JavaScript?


HTML, CSS, JavaScript : They are programming languages to communicate with software. They are used to create website for better user experience.


HTML


HTML stands for Hyper Text Markup Language which is the core of a web site. It works by using tags to identify a variety of contents and it also serves purposes to the webpage as well. Therefore, HTML provides a page structure and written text which can be adjusted and magnified by using CSS and JavaScript.


CSS


CSS stands for Cascading Style Sheets. It is a style information functioning for presentation, formatting, and layout. In other words, CSS controls how the website will look like. The example of CSS are font, color, background, and animatable reference.


JavaScript


Another name for JavaScript is a "client-side" programming language. JavaScript is the language developed by Netscape to create interactive elements to websites. A web page can be more dynamic by using this programming language. Its functions are data calculation, manipulation, and validation. Nowadays, all modern webpage support JavaScript.
